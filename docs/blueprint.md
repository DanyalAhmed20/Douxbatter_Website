# **App Name**: Douxbatter Delights

## Core Features:

- Product Catalog: Grid-based display of baked goods (cakes, brownies, cookies) with category-based navigation and filtering based on dietary preferences and price using the Shopify API.
- Shopping Cart: Persistent cart functionality, allowing users to add, remove, and update items.
- Guest Checkout: Streamlined guest checkout process to minimize friction for new users.
- Saved Payments: Securely save payment options via Stripe/PayPal for faster checkout.
- Order Summaries: Generate order descriptions suitable to send to customer by sms
- Performance Monitoring: Tools to maintain sub 2-second page load times with continuous monitoring.

## Style Guidelines:

- Primary color: Soft brown (#A67B5B) evoking the warmth of baked goods.
- Background color: Light beige (#F5F5DC) to create a clean and inviting canvas.
- Accent color: Muted gold (#D4A27A) for highlighting promotions and interactive elements.
- Font pairing: 'Belleza' (sans-serif) for headlines, and 'Alegreya' (serif) for body text.
- Simple, custom-designed icons for categories and filter options.
- Clean, grid-based layout optimized for mobile devices with clear hierarchy.
- Subtle hover animations and loading indicators to enhance user experience.
